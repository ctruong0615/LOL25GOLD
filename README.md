# League of Legends Gold Difference Analysis
This is a final project for DSC 80 at UCSD which determines how likely a professional League of Legends Esports team is to win a match given gold differences at particular times and the number of drakes/types of drakes slain. Using a variety of different forms of analysis including statistical methods of hypothesis testing, the baseline model of a decision tree, and ________________________________, this projects intends to foster better decision-making within the popular game to improve strategies and win rates.
